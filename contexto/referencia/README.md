# Referência (o seu knowledge base)

Jogue aqui os documentos do seu negócio (`.md`, `.txt`, `.csv`): quem você é, seus
produtos, seu tom de voz, seu público, seus números, seus casos.

O seu Claude Code LÊ esta pasta e escreve os agentes JÁ sabendo do seu negócio, em vez
de genéricos. Quanto mais real o material aqui, mais afiado o time sai.

Você não precisa preencher isto na mão: quando rodar `/setup`, a própria cena deixa você
soltar os arquivos, colar o link do seu site ou trazer o contexto de outro assistente, e
tudo cai aqui. Largar os arquivos aqui antes de rodar também funciona, se preferir.

O que você soltar aqui fica só na sua máquina (não sobe pro Git). Pode apagar este
README quando quiser.
