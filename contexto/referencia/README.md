# Referência (o seu knowledge base)

Jogue aqui os documentos do seu negócio (`.md`, `.txt`, `.csv`): quem você é, seus
produtos, seu tom de voz, seu público, seus números, seus casos.

**Faça isso ANTES de montar o time** (antes de rodar `/config-os`). O seu Claude Code
LÊ esta pasta e escreve os agentes JÁ sabendo do seu negócio, em vez de genéricos.

Quanto mais real o material aqui, mais afiado o time sai. Sem nada aqui, o time nasce
só pelo que você contou na entrevista.

O que você soltar aqui fica só na sua máquina (não sobe pro Git). Pode apagar este
README quando quiser.
